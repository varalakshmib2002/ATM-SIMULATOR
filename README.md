# ATM Simulator – Core Java Project

A simple and user-friendly **ATM Simulation System** built completely using **Core Java (No JDBC)**.  
It demonstrates real-world concepts like **OOP, File Handling, Collections, Exception Handling**, and **Console-based UI**.

This project is perfect for interviews, resumes, and GitHub portfolios.

---

## Features

### ✔ User Login  
- Enter Account ID  
- Enter PIN  
- Validates credentials from `accounts.txt`

### ✔ Check Balance  
Shows current available balance.

### ✔ Deposit  
- Adds money to account  
- Updates balance  
- Writes transaction to `txn_<accountId>.txt`

### ✔ Withdraw  
- Validates sufficient balance  
- Updates balance  
- Records withdrawal in transaction file

### ✔ Mini Statement  
Displays all transactions from the user’s log file.

### ✔ File-based Storage  
- `accounts.txt` → stores account details  
- `txn_<accountId>.txt` → stores user's transactions  
- No database required
